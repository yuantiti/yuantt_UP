第一步

  • 敲入命令：git remote add origin https://github.com/laoda271/Collections.git，如果提示已经存在不是这个远程仓库，可以用git remote rm origin 删除远程仓库地址，然后再加入远程地址： git remote add origin https://github.com/laoda271/Collections.git。如果远程仓库一样则pass

第二步

  • 然后删除文件： git rm README.md

第三步

  • 然后提交文件，git commit -m "删除"

第四步

  • git push -u origin master，等着,会弹出输入用户名，然后再输入密码确认就可以了，之后去github上看就不见了
